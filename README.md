# Time Series Analysis and Inflation Forecasting

This project analyses and forecasts inflation in France and the United States using quarterly data and R.

## Methods
- Time-series diagnostics and stationarity tests
- Structural break and seasonality analysis
- SARIMA forecasting
- ARCH/GARCH volatility modelling
- Johansen cointegration tests
- Cointegrated VAR/VECM forecasting

## Repository Structure
- `Code/` – R scripts
- `Data/` – datasets used in the analysis
- `Project/` – final project report

## Data
Quarterly macroeconomic data are sourced from FRED and cover French inflation, U.S. inflation, and the French 10-year government bond yield.
